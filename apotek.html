FAQ

Pertanyaan Umum

Hal-hal yang sering ditanyakan seputar Nori Apotek.

Bagaimana cara membeli aplikasi ini?
Pembelian bisa melalui link Shopee atau Lynk.id toko kami atau bisa melalui kontak WhatsApp (082112710702) kami langsung.
Apakah ada biaya langganan per-bulan?
Tidak ada, lisensi aplikasi berlaku seumur hidup.
Apakah ada biaya lain selain harga beli aplikasi?
Tidak ada. Harga sudah mencakup semua akses aplikasi. Tidak ada biaya server, domain, bulanan, atau tahunan karena aplikasi bersifat offline.
Apakah aplikasi bisa diakses melalui online?
  • Tidak bisa, karena aplikasi bersifat offline. Namun dapat diakses oleh perangkat lain dalam satu jaringan WiFi melalui IP address perangkat tempat aplikasi terpasang.
  • Cara akses terdapat pada menu Pengaturan Aplikasi → Akses Jaringan Lokal.
Lisensi bisa dipakai di berapa perangkat?
  • 1 lisensi hanya berlaku untuk 1 perangkat.
  • Aktivasi lisensi wajib terkoneksi internet.
  • Jika ingin pindah perangkat, copot lisensi terlebih dahulu melalui menu Pengaturan → Lisensi.
Bagaimana cara migrasi ke perangkat baru?
  • Unduh database dari perangkat lama dan pindahkan ke perangkat baru.
  • Copot lisensi dari perangkat lama melalui menu Pengaturan → Lisensi.
  • Aktifkan lisensi kembali di perangkat baru.
  • Pulihkan database: Pengaturan → Backup & Data → Recover Database.
Adakah tutorial cara pengoperasian aplikasi?
Ya ada, bisa diakses melalui kanal YouTube kami: https://youtube....playlist.
Apa spesifikasi minimum untuk aplikasi ini?
Komputer dengan minimal RAM 4 GB dan sistem operasi Windows 10 ke atas.
Bagaimana cara backup database secara manual?
Melalui menu Pengaturan → Backup & Data → Download Database.
Apa yang terjadi jika lupa backup database?
Aplikasi akan membuat backup otomatis saat pertama kali dibuka dan setiap beberapa jam sekali (aplikasi harus aktif).
Apa yang terjadi jika file database hilang?
Jika hal tersebut terjadi, semua data hilang dan tidak bisa direcover. Ini termasuk human-error sehingga kami tidak dapat membantu pemulihan data.
